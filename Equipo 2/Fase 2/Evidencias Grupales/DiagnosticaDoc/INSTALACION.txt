INSTALACION DEL BACKEND

EN LA RUTA DE LA CARPETA EJEMPLO : 
cd "C:\Users\jaran\OneDrive\Documentos\Proyectos\DiagnosticaDocUp\backend"


 1) Verifica tu versión de Python
python --version
# o
py --version

# 2) Crear el entorno (carpeta .venv)
py -3 -m venv .venv
# (si usas una versión específica)
# py -3.11 -m venv .venv

# 3) Activar
.\.venv\Scripts\Activate.ps1     # PowerShell

# 4) Actualizar pip y instalar deps
python -m pip install --upgrade pip
pip install -r requirements.txt
